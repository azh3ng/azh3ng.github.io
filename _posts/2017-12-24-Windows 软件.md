# 效率软件
在用(好用)的软件

##  [Anki](https://apps.ankiweb.net/)
主要用来背单词

##  [AutoHotKey](https://www.autohotkey.com/)
修改系统快捷键，或者绑定某些快捷键完成自动化操作

### [CapsLockPlus](https://azh3ng.com/2021/06/13/CapsLockPlus.html)
我开源的一个 AutoHotKey 脚本，对键盘的 CapsLock 键增强，极大的提升了键盘效率。

##  [Bandizip](https://www.bandisoft.com/bandizip/)
用起来最顺手的压缩软件

## [Everything](https://www.voidtools.com/zh-cn/)
全局文件搜索工具, 内存占用小, 搜索速度快。  
在功能上与 Listary 有重合  
- 一般搜索文件我会使用 Listary 直接在资源管理器中输入就能找到目标文件.  
- 当要对某种命名规则(前缀/后缀等)的文件做批量处理时, 会用 Everything, 搜索到目标文件, 批量选中操作  

## [nomacs](https://nomacs.org/)
看图软件，启动速度快，支持常见图片格式，适合快速翻看文件夹里的图片  
比系统自带图片查看器更轻量，配合方向键连续浏览很顺手  

## [FireFox](https://www.mozilla.org/firefox/)
常用浏览器  

## [Listary](https://www.listary.com/)
使用频率最高的软件  
全局文件搜索工具, 内存小速度快, 支持中文搜索, [异次元](https://www.iplaysoft.com/) 的 [这篇文章](https://www.iplaysoft.com/listary.html) 介绍的很详细  
我最常用的方式是把启动快捷键改为 `Ctrl + Space`，使用快捷键调起输入框，直接输入文件名，即可打开文件或者定位到文件夹  

## [Geek Uninstaller](https://geekuninstaller.com/)
软件卸载工具，软件卸载后自动检测垃圾文件及注册表，然后一键清除  

## [NotePad++](https://notepad-plus-plus.org/)
NotePad++ 是一个开源的轻量级文本编辑器, 启动速度快, 内存占用小, 文本编辑功能强大, 支持代码高亮, 适合轻量的文本编辑场景.

## [Logseq](https://logseq.com/)
本地优先的双向链接笔记软件，适合用大纲的方式整理笔记

## [Obsidian](https://obsidian.md/)
双向链接笔记软件

## [~~OneNote~~](https://www.onenote.com/)
~~微软出品的笔记软件~~  
已经不使用，笔记主要迁移到 Obsidian

## [1Remote](https://github.com/1Remote/1Remote)
远程连接管理工具，支持 RDP、SSH 等协议

## [Alt-Tab Terminator](https://www.ntwind.com/software/alttabter.html)
替换系统自带的 Alt-Tab 窗口切换，窗口列表更直观

## [Aquile Reader](https://apps.microsoft.com/detail/9p08t4jltqnk)
电子书阅读器，界面干净，适合在 Windows 上看 EPUB

## [PotPlayer](https://www.potplayer.org)
全能视频播放器, 启动速度快, 内存占用小, 配置项丰富, 功能齐全, 免费无广告

## [QuickLook](https://github.com/QL-Win/QuickLook)
文件管理器增强工具，按空格即可预览文件  

## [PixPin](https://pixpinapp.com/)
最常用的截图软件，支持截图、贴图、长截图、OCR、标注等功能  
操作顺手，功能也更完整，日常截图和临时贴图都够用  

## [Sumatra PDF](https://www.sumatrapdfreader.org/)
一款轻量级的 PDF 阅读器。除了支持 PDF 阅读，还支持 EPUB、MOBI 格式的电子书，以及 XPS、DjVu、CHM 等多种格式。  
内存占用小, 启动速度极快。Sumatra PDF 开发者表示「会优先让用户界面更干净」，足以说明它的开发方向。  
适合不需要太多复杂的功能，只想看 PDF 文档或电子书的场景  

## [Syncthing](https://syncthing.net/)
开源文件同步工具，适合多台设备之间同步文件

## [tailscale](https://tailscale.com/)
基于 WireGuard 的组网工具，配置简单，适合远程访问自己的设备

## [火绒](https://www.huorong.cn/)
系统安全工具

## [V2rayN](https://github.com/2dust/v2rayN)
网络代理软件

## [TrafficMonitor](https://github.com/zhongyang219/TrafficMonitor)
Traffic Monitor 是一款用于 Windows 平台的网速监控悬浮窗软件，可以显示当前网速、CPU及内存利用率，支持嵌入到任务栏显示，支持更换皮肤、历史流量统计等功能。

### [Tampermonkey](https://www.tampermonkey.net/)
Tampermonkey 是一款免费的浏览器扩展和最为流行的用户脚本管理器，它适用于 Chrome, Microsoft Edge, Safari, Opera Next, 和 Firefox。
可以对特定网页执行特定脚本, 以满足屏蔽广告, 转换网页样式, 网页增强等功能
- 知乎网页助手
    1、知乎站外链接直接跳转至目标网址；2、自动展开问题全部信息，同时展示所有回答；3、去除知乎网页中的广告；4、知乎网页中短视频下载；5、解除知乎复制限制-划词复制（鼠标左键划词自动添加到剪切板）；6、去除烦人的登录提示；7、自动高清图片显示【注：支持Tampermonke4.0以上版本】【这可能是功能最全面的知乎助手了】
- 护眼脚本
    修改网页背景色，让网页背景色偏白的部分变成乡土黄、豆沙绿，浅色灰还有淡橄榄，更加护眼。默认护眼色是乡土黄。
- 去繁留简-繁简转换
    将页面繁体字转换成简体字显示
- 博客阅读美化（去广告|自动全文|简洁）
    优化特定博客网站的阅读体验: 修改页面布局, 屏蔽广告
- Bilibili Evolved
    哔哩哔哩增强脚本: 下载视频, 音乐, 封面, 弹幕 / 简化直播间, 评论区, 首页 / 自定义顶栏, 删除广告, 夜间模式 / 触屏设备支持
- AC-baidu:重定向优化百度搜狗谷歌必应搜索_favicon_双列
    绕过百度、搜狗、谷歌、好搜搜索結果中的自己的跳转连接，直接访问原始网页


# 开发向软件

## [Git](https://git-scm.com/)
版本管理工具

## [Windows Terminal](https://github.com/microsoft/terminal)
Windows Terminal 是微软开源的终端软件，美观且实用

## [WSL2](https://learn.microsoft.com/windows/wsl/)
WSL（Windows Subsystem for Linux）是可以运行在 Windows 上的 Linux 环境，省去了虚拟机的开销

## [uv](https://github.com/astral-sh/uv)
Python 包管理工具，速度很快，可以替代 pip、venv、pip-tools 等工具  
现在更倾向用 uv 管理 Python 项目环境，命令简单，依赖安装和虚拟环境创建都很快

## [IDEA](https://www.jetbrains.com/idea/)

下面是一些常用的插件
### [.ignore](https://plugins.jetbrains.com/plugin/7495--ignore)
.ignore 文件插件

在kebab-case，SNAKE_CASE，PascalCase，camelCase，snake_case或空格之间轻松切换。快捷键 `⇧+⌥+ U` 或 `Shift + Alt +U`。允许禁用某些转换或在首选项中更改其顺序。

### [Lombok](https://plugins.jetbrains.com/plugin/6317-lombok)
### [Markdown Image Kit](https://plugins.jetbrains.com/plugin/17080-markdown-image-kit)
Markdown Image Kit 是一款在 IDEA 中方便高效得管理 Markdown 文档图片的插件.
### [Mybatis Log](https://plugins.jetbrains.com/plugin/10065-mybatis-log)

### [MyBatisCodeHelperPro](https://plugins.jetbrains.com/plugin/9837-mybatiscodehelperpro)

### [RestfulToolKit](https://plugins.jetbrains.com/plugin/10292-restfultoolkit)
一套 RESTful 服务开发辅助工具集。  
1. 根据 URL 直接跳转到对应的方法定义 ( Ctrl \ or Ctrl Alt N );
2. 提供了一个 Services tree 的显示窗口;
3. 一个简单的 http 请求工具;
4. 在请求方法上添加了有用功能: 复制生成 URL;,复制方法参数...
5. 其他功能: java 类上添加 Convert to JSON 功能，格式化 json 数据 ( Windows: Ctrl + Enter; Mac: Command + Enter )。
### [Translation](https://plugins.jetbrains.com/plugin/8579-translation)
翻译插件，快捷键`Ctrl + Shift + Y`

### [generateAllSetter](https://plugins.jetbrains.com/plugin/9360-generateallsetter)
